# Django Account Management

## Installation

```
pip install django-account-management
```

add `account_management` to your `INSTALLED_APPS` in `settings.py`

```python
INSTALLED_APPS = [
    ...
    "account_management",
    "flex_menu",
    "easy_icons",
    ...
]
```
